Additions

  • Window, frame and dialog step types implemented for playback and export.
  • JUnit and TestNG export.
  • Ability to put escape sequences like \n into step parameters.
  • Pause implemented for Selenium 1 RC playback.
  • Can now reboot Builder from the plugins manager.

Fixes

  • Fixed bug where dialogs would not pop up in Firefox preferences.
  • Fixed bug where invalid security certificate warnings would fail to appear.
  • Failed negated assert steps now fail "hard" like asserts, not "soft" like verifies.
  • Improved shutdown reliability.
  • Improved PHP exporter.
  • Various other small fixes.
