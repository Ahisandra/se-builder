As simple as it gets

1

Enter a URL & choose Selenium 1 or 2.
2

Click user actions you want to test.
3

Run tests locally or on Selenium RC.
Save. Export in your programming language of choice. You're done.

Selenium 1 & 2

The goal of the project is to support both Selenium 1 and 2 using a simple JSON based format we have created, allowing scripts to be exported and imported easily.

Extensibility

We're developing a plugin framework to make it easy to add functionality to Builder. Plugins can be installed and managed directly from inside Builder.
